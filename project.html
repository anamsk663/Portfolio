Tic Tac Toe

The Tic Tac Toe game is a simple two-player strategy game developed as a beginner-level programming project. It involves a 3x3 grid where players take turns marking their symbol (X or O) with the goal of placing three of their marks in a horizontal, vertical, or diagonal row. The project helps reinforce fundamental programming concepts such as loops, conditionals, arrays (or lists), and user input. It can be implemented with a text-based interface or a graphical user interface (GUI), and may include additional features like score tracking or AI opponent logic.
